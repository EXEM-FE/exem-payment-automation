# 10. 참고 문서 / 데이터 카탈로그

> 설계·구현 시 참조해야 할 외부 문서, 사내 가이드, 실 데이터의 위치와 의의를 한곳에 정리.

## A. 사내 공식 규정 / 가이드

이 문서들은 룰 엔진과 화면 카피의 source of truth. 갱신 시 변경점 추적 필요.

### A1. 2026년 경비규정.pdf 🌟 (룰 엔진의 핵심 출처)

| 항목 | 내용 |
|---|---|
| 위치 | `~/Downloads/2026년 경비규정.pdf` |
| 크기 | 약 330KB |
| 발행 | 주식회사 엑셈 (재무회계팀) |
| 영향 | `rules/*.json`의 한도·요율·마감일 거의 전부 |

**핵심 섹션 (1~5p 확인)**
- §I-1 경비 결제: 법인카드 / 카드 불가 항목(통행료·주차) 현금 / 세금계산서 송금
- §I-2 경비 청구: **매월 7일(영업일 기준)** 비즈웨어 전자결재
- §I-3 현금: 익월 7일까지 오프라인
- §I-4 세금계산서: 지급 3일 전까지
- §II-1 경조금: 결혼/회갑/출산/조의 표
- §II-2 선물대: 직급별(대표 100만 ~ 사원 20만, 생일 3만)
- §II-3 건강검진: 50만원/2년/1회
- §II-4 통신비: 임원·영업 실비 / 컨설팅 75,000 / 개발·일반 65,000

> ⚠ 1차 룰 JSON에 이 5페이지의 표를 우선 옮긴다. 나머지 페이지는 2차에 점진 반영.

### A4. 개인법인카드 경비신청방법.pdf 🌟 (구현 시 화면별 step-by-step 출처)

| 위치 | 사본 1: `~/Downloads/개인법인카드 경비신청방법.pdf` (또는 `(1)` 사본)<br>사본 2: `.context/attachments/개인법인카드 경비신청방법 (1).pdf`<br>페이지 PNG 36장: `.context/attachments/개인법인카드 경비신청방법 (1)_page_*.png` |
|---|---|
| 분량 | 36 슬라이드 |
| 용도 | 카드 신청 절차의 정답지 |

**핵심 섹션**
- §1 결재 준비 (3~5p): 현대카드 MY COMPANY 가입, 양식 다운로드
- §2 양식 작성 (6~21p)
  - 7~10p: 명세서 다운로드 → A2:N5 복사 → A3 붙여넣기
  - 10p: 계정 자동 추천 예시
  - 11p: **14개 계정 종류** (복리후생비/회식비/어회교통비/접대비/회의비/소모품비/사무용품비/통신비/운반비/지급수수료/광고선전비/도서인쇄비/교육훈련비/잡비)
  - 12p: 경비신청금액 입력 시 개인사용금액 자동 계산
  - 13p: 식대(야근/점심/출장) 작성 시 **모든 사용자 기재 필수**, 타 부서 인원 결제 건은 경비 인정 X
  - 14p: 주유비 = `거리(km)*편도/왕복*이용일수`, 1km=300원 기준
  - 15p: 취소수수료 (업무상=여비교통비, 개인사정=개인사용)
  - 16~17p: 인터넷쇼핑몰/편의점/배달앱 영수증 첨부 (사진 또는 매출전표)
  - 18~21p: 후불교통/하이패스 명세서 처리 (1차 비목표)
- §3 그룹웨어 (22~36p): 다우오피스 기안문 작성·결재
