# 01. 비전 / 문제 정의

## 페르소나

**최기환, FE1팀 프론트엔드 개발자.**
- 매월 1회 법인카드 경비 신청
- 식대(야근/회식)가 80%, 가끔 주유·택시
- 카카오톡으로 받은 영수증 사진을 갤러리에 모아둠
- 현재는 30~60분 손작업 후 그룹웨어(다우오피스)에 기안

## 현재의 진짜 고통 (단순 "엑셀이 귀찮아"가 아님)

| 고통 | 진짜 원인 |
|---|---|
| 30~60분 손작업 | 명세서 14컬럼 복붙 + 영수증 정렬 |
| **"누구랑 먹었더라?"** | 한 달 후 기억해야 함 |
| **영수증 분실** | 그 자리에서 안 찍어둠 |
| 룰 위반 (한도/품의서/영수증 누락) | 규정이 길어서 모름 |

→ **가장 큰 두 가지(기억 망각, 영수증 분실)는 "그때그때 기록"으로만 해결된다.**
손작업 30분을 줄이는 건 부수효과지, 본질이 아니다.

## 가치 제안

> **식사 자리에서 폰으로 30초**, 월말에 **PC에서 2분 검토 → 다운로드**.
> 기억이 신선할 때 적고, 영수증은 그 자리에서 사진. 한 달 후의 추리 게임을 없앤다.

## 비목표 (1차에서 안 함)

- ❌ 일반경비(현금) 종이 양식 — 흐름 자체가 다름
- ❌ 품의서 작성 자동화 — 50만원 초과시 "경고만" 표시
- ❌ 사외교육·자격증 신청서 — 그룹웨어 별도 양식
- ❌ 출장비/일비 자동 계산 — 룰 엔진 확장 시 2차
- ❌ 회식비 월/연 누적 한도 트래킹 — 데이터 누적 시점 이후
- ❌ 다우오피스 멤버 자동 동기화 — API 키 발급 후 2차
- ❌ 명세서 자동 다운로드(현대카드 사이트 크롤링) — 사용자가 직접 받음
- ❌ 그룹웨어 자동 기안 — 사용자가 다운로드 후 직접 첨부

## 시간 절감 가설

| 단계 | 현재(수기) | 앱 사용 후 |
|---|---|---|
| 명세서 복붙 | 5분 | **0분** (자동 파싱) |
| 행별 입력 7건 | 15분 | **3분** (저널에 이미 90% 기록돼 있음) |
| 영수증 정렬·붙임 | 20분 | **2분** (자동 매칭) |
| 검토 | 5분 | **2분** (미리보기) |
| **합계** | **45분** | **7분** |

월말 PC 작업 외에, 일상에서 식사·구매 자리에서 30초씩 N회 추가 입력 비용이 발생하지만, 이는 "그 자리에서만 알 수 있는 정보(누구랑/무슨 일로)"를 캡처하는 가치가 충분히 보상한다.

## 기준 사용자 수 / 사용 빈도

- 1차: FE1팀 ~10명
- 사용 빈도: 1인당 월 평균 7~13회 입력 + 월 1회 정산
- 월 데이터 부피: 1인당 사진 ~25장(1600px JPEG, 5MB) + 항목 ~13건
